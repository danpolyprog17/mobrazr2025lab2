#### 4.2. –õ–æ–∫–∞–ª—å–Ω–æ–µ —Ö—Ä–∞–Ω–∏–ª–∏—â–µ (AsyncStorage)

**–£—Ç–∏–ª–∏—Ç–∞ –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å —Ö—Ä–∞–Ω–∏–ª–∏—â–µ–º (`utils/storage.js`):**

    return false;

    return null;

/**

      data,

    const age = Date.now() - cached.timestamp;

      return null;
    }

    return cached.data;
  },
