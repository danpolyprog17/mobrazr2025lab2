## Лабораторная работа №2 — макет мобильного финансового трекера

Это демо-приложение на **React Native (Expo)**, созданное в рамках лабораторной работы по теме «Создание основного макета мобильного приложения».

### Функциональность
- Пять разделов, управляемых через ручную таб-навигацию (без внешних библиотек):
  - **Обзор** — карточка общих расходов и список последних операций.
  - **Категории** — прогресс-бар по каждому бюджету.
  - **Лидерборд** — сравнение друзей по «баллам экономии».
  - **Блог** — карточки образовательных постов.
  - **Профиль** — карточка пользователя со статистикой.
- Все данные заглушечные, но структура повторяет реальные сценарии финансового трекера.

### Технологии
- [Expo](https://expo.dev/) (React Native)

### Установка и запуск
```bash
git clone https://github.com/<ваш-аккаунт>/lab2-app.git
cd lab2-app
npm install
npm start
```

Далее можно:
- открыть приложение через Expo Go (Android/iOS);
- запустить эмулятор Android: `npm run android`;
- запустить веб-версию: `npm run web`.

### Структура проекта (основное)
- `App.js` — главный файл приложения, содержит таб-навигацию и все экраны макета.
- `REPORT.md` — текст отчёта, который можно конвертировать в PDF.

### Как получить PDF-отчёт
1. Откройте файл `REPORT.md` в редакторе (например, VS Code).
2. Используйте расширение или утилиту (например, `pandoc`) для экспорта в PDF:

```bash
pandoc REPORT.md -o REPORT.pdf
```

3. Проверьте форматирование PDF и при необходимости подправьте стиль в `REPORT.md`.
